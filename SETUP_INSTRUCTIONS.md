# Setup and Run Instructions

## Quick Setup (5 minutes)

### 1. Clone Repository

```bash
git clone https://github.com/RBATHINENI78/AI-Powered_Vacation_Planner.git
cd AI-Powered_Vacation_Planner
```

### 2. Create Virtual Environment

```bash
# Create virtual environment
python3 -m venv venv

# Activate it
source venv/bin/activate  # On macOS/Linux
# OR
venv\Scripts\activate     # On Windows
```

### 3. Install Dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Set Up Environment Variables

```bash
# Copy example env file
cp .env.example .env

# Edit .env with your API keys
# You need:
# - GOOGLE_API_KEY (Get from: https://makersuite.google.com/app/apikey)
# - OPENWEATHER_API_KEY (Get from: https://openweathermap.org/api)
# - EXCHANGERATE_API_KEY (Get from: https://www.exchangerate-api.com/)
```

**Edit `.env` file:**
```bash
# Google Cloud
GOOGLE_API_KEY=your-gemini-api-key-here

# External APIs (Free Tier)
OPENWEATHER_API_KEY=your-openweather-key
EXCHANGERATE_API_KEY=your-exchangerate-key

# Database
DATABASE_PATH=./data/vacation_planner.db

# MCP Server
MCP_SERVER_PORT=3000

# Settings
ENVIRONMENT=development
LOG_LEVEL=INFO
```

### 5. Run the Application

```bash
# Make sure you're in the project root directory
cd AI-Powered_Vacation_Planner

# Run the application
python src/main.py
```

## Troubleshooting

### Issue: "GOOGLE_API_KEY not set"

**Solution:**
1. Make sure you created `.env` file from `.env.example`
2. Add your Gemini API key to `.env`
3. Get API key from: https://makersuite.google.com/app/apikey

### Issue: "Module not found"

**Solution:**
```bash
# Make sure you activated virtual environment
source venv/bin/activate

# Reinstall dependencies
pip install -r requirements.txt
```

### Issue: "google.generativeai not found"

**Solution:**
```bash
# Install specific version
pip install google-generativeai --upgrade
```

### Issue: Permission denied

**Solution:**
```bash
# On macOS/Linux, you may need to give execute permission
chmod +x src/main.py
```

## Getting API Keys

### Google Gemini API (FREE)
1. Go to: https://makersuite.google.com/app/apikey
2. Click "Create API Key"
3. Copy and paste into `.env`

### OpenWeather API (FREE Tier)
1. Go to: https://openweathermap.org/api
2. Sign up for free account
3. Get API key from dashboard
4. Free tier: 60 calls/minute

### ExchangeRate API (FREE Tier)
1. Go to: https://www.exchangerate-api.com/
2. Sign up for free account
3. Get API key
4. Free tier: 1,500 requests/month
